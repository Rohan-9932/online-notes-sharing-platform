# Online Notes Sharing Platform 🚀
[![Live Demo](https://img.shields.io/badge/Demo-Live%20Project-green?style=for-the-badge&logo=github)](https://github.com/Rohan-9932/online-notes-sharing-platform)

A full-stack web application designed for students and creators to upload, manage, and download academic resources securely.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Modern Responsive Layout)
* **Backend:** Node.js, Express.js
* **Database:** MySQL
* **File Handling:** Multer Middleware

## 📌 Core Modules Completed
1. **Dynamic Dashboard:** Live data fetching from MySQL straight to note cards.
2. **User Upload Pipeline:** Secure PDF upload with local server storage.
3. **Admin Control Panel:** Moderation tools for user management and database logs.
4. **Note Details View:** Parameterized SQL filtering with email lead-generation modals.
