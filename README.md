# 📚 Online Notes Sharing Platform

A secure, full-stack web application designed for students to seamlessly upload, share, and manage academic notes, organized by subjects and semesters. It also includes features for safe user authentication and integrated payment tracking for premium notes.

## 🚀 Live Demo

Experience the fully functional deployment live on the web:
🔗 **[Live Website Link](https://online-notes-sharing-platform.onrender.com)**

---

## 🛠️ Tech Stack & Architecture

- **Backend:** Node.js, Express.js
- **Database:** MySQL (Hosted securely on Clever Cloud)
- **Authentication:** Password hashing with `bcrypt` & session management via `express-session`
- **File Uploads:** Managed smoothly using `multer`
- **Deployment:** Render (Web Service)

---

## ✨ Features

- **Secure Authentication:** Robust Signup and Login system using encrypted password storage.
- **Dynamic Notes Dashboard:** View and download shared notes filtered by semester and category.
- **Protected File Uploads:** Upload academic resources safely with automatic file tracking.
- **Session Protection:** Restricts unauthorized users from uploading or modifying critical data.
- **Demo Payments:** Features an integrated demo payment logs structure for premium note transactions.
